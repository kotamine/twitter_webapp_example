This site is a simple web application of Twitter data streaming and analysis, powered by Twitter API, Python, Flask, and JavaScript.

It is based on an online tutorial with some modifications.

Languages page samples from all Twitter streaming data (say, 1,000 tweets) and displays the language compositions of all tweets and top tweets (for example, those with over 10,000 retweets) in pie charts.

Top Tweets page shows the top tweets as defined in the Languages page.

Trends page calls Twitter's trend API for all locations (trends_place(1)) and fetches 3 popular tweets for each trending word. This page is updated less frequently than Languages and Top Tweets pages in order to stay under Twitter's API request limit rate.

* Please note that this web app may stop or pause if it accidentally reaches the API request limit rate.
